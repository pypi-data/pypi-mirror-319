# AggregationFrequency

The time period used to group data in the response. Note that this is only valid for the `PERFORMANCE` time period type.

## Enum

* `WEEK` (value: `'WEEK'`)

* `MONTH` (value: `'MONTH'`)

* `QUARTER` (value: `'QUARTER'`)

* `YEAR` (value: `'YEAR'`)
