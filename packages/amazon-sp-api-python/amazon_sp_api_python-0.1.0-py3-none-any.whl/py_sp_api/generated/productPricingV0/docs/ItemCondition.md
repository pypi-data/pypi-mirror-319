# ItemCondition

Filters the offer listings to be considered based on item condition. Possible values: New, Used, Collectible, Refurbished, Club.

## Enum

* `NEW` (value: `'New'`)

* `USED` (value: `'Used'`)

* `COLLECTIBLE` (value: `'Collectible'`)

* `REFURBISHED` (value: `'Refurbished'`)

* `CLUB` (value: `'Club'`)
