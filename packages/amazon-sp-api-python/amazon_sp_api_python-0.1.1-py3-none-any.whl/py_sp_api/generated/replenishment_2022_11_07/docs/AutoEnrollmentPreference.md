# AutoEnrollmentPreference

The auto-enrollment preference indicates whether the offer is opted-in to or opted-out of Amazon's auto-enrollment feature.

## Enum

* `OPTED_IN` (value: `'OPTED_IN'`)

* `OPTED_OUT` (value: `'OPTED_OUT'`)
