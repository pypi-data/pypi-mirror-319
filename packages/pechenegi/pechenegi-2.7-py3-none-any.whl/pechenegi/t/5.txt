Цикл for используется для итерации по коллекциям (списки, строки, диапазоны). Он позволяет выполнять действия для каждого элемента коллекции. Функция range() генерирует последовательность чисел и часто используется для итерации по числовым диапазонам. Функция enumerate() позволяет итерировать по коллекции, возвращая индекс и значение каждого элемента.
