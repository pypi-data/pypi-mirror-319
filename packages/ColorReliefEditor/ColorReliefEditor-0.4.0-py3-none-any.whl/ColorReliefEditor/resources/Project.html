This tab creates and opens Color Relief projects. A project stores all the hillshade and color settings needed to create a relief image

Create a Project

  1. Click New. A dialog will appear to create and then open the project folder. A sample color ramp and settings file will be copied to this folder.

Create an image

Follow these steps to generate your relief image:

  1. Select the Elevation tab to download and configure Digital Elevation files.
  2. Adjust and preview hillshade and color settings in the Hillshade and Color tabs.
  3. Finally, use the Create tab to produce a full-sized merged image, combining the hillshade and color relief.

Modes

Basic Mode:
  • A simplified interface for common settings.
Expert Mode:
  • Offers control over all settings
  • Supports the use of multiple layers, each with its own elevation files but sharing hillshade and color relief settings.

Optionally provides two additional tabs for access to every setting by setting "Tabs" to Extended in the Project window:

Misc: Rarely changed GDAL settings.(Only in expert/extended mode)

Settings: Update general application settings.(Only in expert/extended mode)

What is a Project?

A project defines the style for the relief image including all color settings, hillshade What is a Layer?

A layer is a set of contiguous elevation files with the same resolution. In basic mode, a project just has one layer. In expert mode, a project can have multiple layers all using the same project style. One use for layers is to have higher resolution images with the same style, such as a layer for US, and a higher resolution layer for Washington, and a higher resolution layer for Mt Rainier. Another is to have separate, discontinuous layers such as a layer for Mt Rainier and a layer for Mt Saint Helens.
