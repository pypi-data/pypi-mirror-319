This tab lets you modify the color for each elevation and immediately preview the result.

Instructions

Modify Elevation: Click on an elevation value to edit it.

Change Color: Click on a color to the right of the elevation to modify it.

Preview: Click Preview to generate a small preview for the current settings.

Rescale: Set a new maximum elevation and rescale all rows.

Drop File: Drag and drop an existing GDAL Color file to the blue square. NOTE - This will overwrite the existing color file!

Buttons

  • Insert: Click an elevation, then click Insert to add a new row above it.
  • Delete: Click an elevation, then click Delete to remove that row.
  • Undo: Reinstate previous values. The undo history is cleared when the application exits.
