This tab lets you tweak hillshade settings and immediately preview the result.

Instructions

  • Shading: Choose the shading algorithm.
  • Strength: Adjust the strength of the shading.
  • Preview: Click Preview to generate a small hillshade preview with the current settings.

Expert Settings

  • Other: Set any other gdaldem hillshade settings such as -s 3 (scale), -alt 3 (altitude)
  • Not all parameter combinations are valid. Refer to gdaldem hillshade documentation for more details.
  • Ensure parameters are correctly formatted:
  • Strength: -z 3
  • Scale: -s 3
  • Altitude: -alt 3

Notes

The first time the preview is built it will take considerably longer
