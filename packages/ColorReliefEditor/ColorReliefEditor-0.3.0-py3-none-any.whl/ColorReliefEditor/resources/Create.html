This tab creates full-size hillshade and color images and then merges them into a relief image.

Instructions

Create: Click Create to generate the final full-size image for this layer.

View: Launches an external viewer to display the full-size image. The default system viewer is the default but QGIS is highly recommended instead. You can change this in the expert mode Settings tab

Expert Settings

  • Calc: The calculation for combining the hillshade (A) and color (B) layers. The default is a composite multiply.
  • Quiet Mode:
  • -q will show minimal output.
  • -v show detailed progress
  • --version show detailed progress and gdal version

Publish: Clicking Publish will copy the image file for the current layer to the destination folder.

Publish To: The destination folder to copy the image to when Publish is clicked. Click on field and enter full path.

Delete Temp: This will delete intermediate files for this layer. Use this when you are done with a project to free space. The intermediate files significantly speed up image creation.



Use the Elevation Tab to switch the active layer

If you have multiple layers, you need to run Create and Publish for each layer.

Notes

The output window is mainly to show progress. You only need to read the details if there is a problem.

The app will display 'Done' when the image is successfully built. Click View to view it externally.

"Create" runs multiple processes in parallel, so the output may be garbled.
