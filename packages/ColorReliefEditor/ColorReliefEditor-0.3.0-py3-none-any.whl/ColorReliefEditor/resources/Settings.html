This tab manages general application settings.

Settings

  • Download: URL for a site to download Digital Elevation Files
  • US High Res Download: URL for a site to download US High Resolution files
  • Viewer: Select the app to use to view images.
  • Multiprocessor:Whether to use multiprocessors (faster). The only downside is the output is garbled, which makes troubleshooting difficult.
  • Verbose: Verbosity level. 0 is minimal, 2 is detailed
  • Font Size: Select the fontsize for the app.
